Generative Design for Structural Topology Optimisation

cGANDeep LearningFEA ABAQUSMATLAB

An M.Tech project combining machine learning with finite-element analysis to generate optimal structural layouts in seconds instead of hours.

Project overview

This project introduces a framework that leverages machine learning to accelerate structural topology optimisation. The goal was a generative model that predicts an optimal material layout for a given set of boundary conditions and loads — drastically reducing the computational time required by traditional iterative FEA.

Concept image coming soon

Methodology & tools

A large dataset of optimised topologies was generated using ABAQUS and MATLAB, then used to train a Conditional Generative Adversarial Network (cGAN). The trained model produces a highly efficient structural design in seconds — a process that previously took hours of simulation. Results were validated by running FEA on the generated designs to confirm structural integrity.

Key achievements

  • Reduced design computation time by over 95%.
  • Predicted topologies with 98% structural-performance accuracy.
  • Built a framework generalisable across a variety of structural problems.
