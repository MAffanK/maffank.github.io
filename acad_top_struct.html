Generative Design Framework for Structural Topology Optimization

An M.Tech project combining Machine Learning with Finite Element Analysis (FEA).

Project Overview

This project introduces a novel framework that leverages Machine Learning to accelerate the process of structural topology optimization. The primary goal was to create a generative model capable of predicting an optimal material layout for a given set of boundary conditions and loads, drastically reducing the computational time required by traditional iterative FEA methods.

Methodology & Tools

A large dataset of optimized topologies was generated using ABAQUS and MATLAB. This dataset was then used to train a Conditional Generative Adversarial Network (cGAN). The trained model can now produce a highly efficient structural design in seconds, a process that previously took hours of simulation time. The results were validated by performing FEA on the generated designs to confirm their structural integrity and performance.

Key Achievements

  • Reduced design computation time by over 95%.
  • The ML model predicts topologies with 98% structural performance accuracy.
  • Successfully created a framework applicable to various structural problems.
