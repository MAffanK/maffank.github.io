CFD–Machine Learning Pipeline for Turbine Optimization

A hybrid approach combining Computational Fluid Dynamics (CFD) and Machine Learning for optimizing hydraulic turbine blade geometry. Developed as an independent project to accelerate turbine design through data-driven surrogate modeling.

Dynamic Meshing CFD Simulation

The project began with detailed ANSYS Fluent simulations using Dynamic Meshing to analyze the transient fluid–structure interaction around turbine blades. Each simulation captured pressure, velocity, and torque over time for different blade geometries and flow rates.

The obtained torque and power outputs were validated using velocity triangle theory, ensuring that CFD results aligned with analytical expectations and experimental trends.

This CFD dataset formed the foundation for training a surrogate model that could predict efficiency rapidly without repeated full simulations.

Velocity contour animation from dynamic mesh simulation

Machine Learning–Based Surrogate Modeling

Using data from 50+ simulation runs, features such as blade angle, curvature ratio, and flow coefficient were extracted. A regression model trained on this data could predict turbine efficiency with an accuracy exceeding 95%.

The trained model replaced CFD in subsequent optimization runs, enabling rapid exploration of new blade designs using the surrogate’s predictions.

The resulting pipeline achieved a 10× reduction in computational cost, while maintaining near-CFD accuracy — a step toward intelligent, data-driven turbine design.

Key Tools & Techniques

ANSYS Fluent (Dynamic Mesh, UDFs) • Python (scikit-learn, NumPy, Pandas) • Data-driven Surrogate Optimization • CFD Post-Processing & Validation via Velocity Triangle Theory
